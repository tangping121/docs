# 幼小衔接 · 每日学习记录

> 基于"幼小衔接家庭指导方案"的每日学习记录与追踪应用，帮助家长科学衔接、轻松记录。

## 项目简介

这是一个纯前端的 Web 应用，无需后端服务，数据存储在浏览器本地（localStorage），支持导出/导入 JSON 备份。

基于前期制定的**幼小衔接家庭指导方案**，将方案中的六大项目、各阶段任务、每周检查表、专注力评估工具整合为一个可交互的日常管理工具。

## 功能模块

### 📊 今日概览
- 每日打卡：按项目分类，勾选今天完成的项目
- 每日笔记：记录观察、进步、调整方向
- 统计卡片：今日完成数、总体进度、连续记录天数、本周检查完成数

### 📁 项目管理
六大项目，每个项目下有多个子阶段：
1. 🎯 习惯养成 — 作息规律、专注力训练、自理能力
2. 📖 阅读与识字 — 亲子阅读、生活识字、拼音启蒙
3. 🔢 数学与逻辑 — 数感、运算启蒙、空间方位、时间认知、逻辑思维
4. ✏️ 书写准备 — 线条练习、形状描红、汉字描红
5. 🗣️ 表达与社交 — 自我表达、社交能力、抗干扰训练、等待专注
6. 💖 心理建设 — 期待小学、抗挫力、入学准备

### 📅 日历视图
- 月历展示，标记有记录的日期
- 点击日期查看当日完成详情和笔记

### ✅ 周检表
- 11项每周自检项，对应方案中的家长每周自检表
- 按周自动归档

### 🎯 专注力评估
- 5维度专注力迁移评估（安静专注/干扰专注/群体专注/无关注自处/等待表现）
- 每项三级评分：❌ 还没准备好 / ⚠️ 基本可以 / ✅ 准备好了
- 3项以上"✅ 准备好了"说明环境迁移基本完成

### 📋 历史记录
- 按时间倒序查看每日记录

### 📥📤 数据管理
- 导出：一键下载 JSON 备份文件
- 导入：从 JSON 文件恢复数据

## 技术栈

- 纯 HTML + CSS + JavaScript
- 无框架依赖，单文件应用
- 数据存储：localStorage
- 响应式设计，支持移动端

## 本地运行

```bash
# 方式一：直接打开
open index.html

# 方式二：使用 http-server
npx http-server . -p 3000 -o

# 方式三：Python
python -m http.server 3000
```
